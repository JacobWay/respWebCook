The first step is to create a very simple blog style page with comments embedded in the div element.

Comments

Under the .comment title, you will add your first comment. And next, inside that comment, immediately after the closing paragraph tag add a comment to that comment.

Continuing from there, you can insert more comments the same way to a comment on the parent comment, or add a comment outside of the parent div element to make the comment to the parents' parent, all the way up to the original blog post.
